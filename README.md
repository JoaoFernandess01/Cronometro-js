##Cronômetro JS
Este projeto consiste em um cronômetro desenvolvido com HTML, CSS e JavaScript puro. A aplicação permite iniciar, pausar e resetar a contagem do tempo, além de registrar marcações em tempo real exibidas dinamicamente em uma lista ao lado do cronômetro. Durante o desenvolvimento foram praticados conceitos importantes de manipulação do DOM, eventos, controle de estados dos botões, uso de setInterval, modularização de funções e estilização com CSS, incluindo personalização da scrollbar e organização responsiva da interface.
